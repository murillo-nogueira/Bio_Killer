# 📖 Sobre o Projeto

O **Bio Killer** é um jogo de plataforma 2D onde o jogador precisa sobreviver enfrentando diferentes inimigos, coletando recursos e avançando pelas fases.

Durante a partida, o jogador controla o personagem principal, desvia dos inimigos, coleta vidas e munições e elimina ameaças utilizando disparos. Conforme o progresso, novos obstáculos e inimigos tornam o jogo mais desafiador.

Este projeto foi desenvolvido em equipe durante a graduação, aplicando conceitos de desenvolvimento web, programação orientada a objetos em JavaScript e metodologias ágeis.
