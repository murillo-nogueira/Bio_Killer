# 📖 Sobre o Projeto

O **Bio Killer** é um jogo de plataforma 2D desenvolvido em equipe como projeto acadêmico, utilizando HTML5, CSS3 e JavaScript puro.

Durante a partida, o jogador controla o personagem principal, enfrenta diferentes inimigos, coleta recursos e avança pelas fases do jogo. O projeto foi planejado e desenvolvido de forma colaborativa, aplicando conceitos de desenvolvimento web, lógica de programação e trabalho em equipe.

---

# 🛠️ Ferramentas Utilizadas

## 🎨 Figma

Toda a interface e o planejamento visual do jogo foram desenvolvidos no **Figma**, permitindo que a equipe definisse layouts, elementos gráficos e a experiência do usuário antes da implementação.

---

## 📋 Trello

O gerenciamento do projeto foi realizado utilizando o **Trello**, organizando o fluxo de trabalho através de:

- Product Backlog
- Sprint Backlog
- Em desenvolvimento
- Revisão
- Concluído

Essa organização permitiu acompanhar o progresso das atividades e facilitar a comunicação entre os integrantes da equipe.

---

# 🔄 Metodologia

O projeto foi desenvolvido utilizando a metodologia ágil **Scrum**, promovendo uma organização eficiente das tarefas e acompanhamento contínuo do desenvolvimento.

As atividades foram divididas em Sprints, permitindo planejamento, acompanhamento e entregas graduais durante todo o projeto.
